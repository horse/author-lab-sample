# Contributing

<!-- 这是一个 sample，文件实质完成后删掉这行注释 -->

Use a focused branch and pull request. Describe the evidence or contract changed, run repository validation, and avoid mixing source-corpus changes with author-model or publication changes in one commit unless the dependency is explicit.

Research changes must include evidence identifiers. Schema changes must update examples and tests. Derived-author changes must update lineage or derivation records when inheritance changes.
