# Source Research Stage Prompt

<!-- 这是一个 sample，文件实质完成后删掉这行注释 -->

Required policy rule: `POLICY-PROVENANCE-001`.

Build the work-item research pack from traceable task sources. Separate verified facts, reported claims, official claims, analysis inferences, literary constructions, and unresolved questions. Record dates, as-of scope, source identifiers, and contradictions.

This stage researches the current writing topic; it must not create new claims about the source author from derived-author output. Source-author research belongs in `source-author-research/` and must use registered versioned segment IDs.
