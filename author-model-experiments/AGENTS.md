# Author Model Experiment Instructions for Agents

<!-- 这是一个 sample，文件实质完成后删掉这行注释 -->

Experiments may reference canonical work items and evaluation results but must not rewrite them. Writing agents must never load evaluator-only held-out content. Condition labels must be hidden from blind evaluators, runtime and model settings must remain controlled, and failed or excluded runs must remain recorded rather than deleted.

Required policy: `POLICY-HELDOUT-001`.
