# Author Model Experiments

<!-- 这是一个 sample，文件实质完成后删掉这行注释 -->

An experiment is a first-class object that connects one hypothesis, controlled inputs, multiple conditions, reproducible run records, blinded evaluation, failure cases, aggregate analysis, and a conclusion. Work items generate individual outputs; evaluations define measurement; experiments explain why and how conditions are compared.

Real held-out prompts and evaluator materials must use an `evaluator-storage://` URI and remain outside writer-readable repository storage under `POLICY-HELDOUT-001`.
